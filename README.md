# DataStructures_snakeGame

Простая игра Змейка

Инструкция:

для перемещения вверх нажмите «W»

для перемещения вниз нажмите «S»

для перемещения левой кнопкой мыши «A»

для перемещения мыши правой кнопкой мыши «D»

для паузы нажмите «Q»

для перехода нажмите «E»
